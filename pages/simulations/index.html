# Simulations

これまでに作ったシミュレーションとかのまとめ。

Epidemic Simulator

感染症が人を媒介してどのように広まるかを可視化した簡易モデル。

円周率プール

G.Galperin (2003) の "Playing Pool with PI" を可視化したもの。 比重が異なる２つの積み木を弾性衝突させることで円周率を任意の桁数まで算出することが出来る。

Ising模型 / XY模型

モンテカルロ法を用いた２次元および３次元のイジング模型シミュレーション。 スピンが任意の角度を向くXY模型も実装している。

２重振り子

２つの振り子が連結している２重振り子のシミュレーション。 カオス運動の一例。

量子力学あれこれ

トンネル効果を始めとした量子力学的現象のシミュレーションを集めたリポジトリ。 主にシュレディンガー方程式を有限差分法で解いているだけなので初学者向け。

ブラウン運動

ランジュバン方程式によるブラウン運動のシミュレーション。

２次元 変形原子核

変形原子核の２次元射影。

３次元 変形原子核

Hill-Wheeler方程式を用いた変形原子核の３次元プロット。
