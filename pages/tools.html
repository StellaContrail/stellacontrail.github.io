# Tools

これまでに作ったツールとかのまとめ。

Amazon Price Recorder

Amazonに出品されている商品の価格変動を記録するツール。 価格が設定した閾値を下回った時にメール通知する機能付。 ウェブスクレイピングの明示的な禁止ルール策定により開発停止。

GIFPlayer

GIF画像をコマ送り出来る再生プレイヤー。 表示しているフレームの画像だけ保存したり出来る。 メモリに全フレーム情報を保存することによって高速なコマ送りが可能に。

コミケ参加者を一挙検索するやつ

Twitterのフォロー覧からコミケに参加する人を抽出し、どこのブースでいつ販売しているのかを一覧化するツール。 Twitter APIの有料化に伴い開発停止。

BeatStreamっぽい音ゲー

画面の左右上下からノーツが降ってくる音ゲー。Unityの勉強が目的だったが、割とその他のタイミング管理等々が難しかった。
